## About The Repository
This repository contains real-world Data Analysis projects focusing on Exploratory Data Analysis (EDA), Statistical Modeling, and powerful Data Visualizations. 

### Key Capabilities Demonstrated
- **Data Cleaning & Preprocessing:** Handling missing values, outlier detection, and data normalization.
- **Statistical Inference:** A/B testing, hypothesis testing, and probability distributions.
- **Exploratory Data Analysis (EDA):** Deep dives into multi-dimensional datasets to extract actionable insights.
- **Data Visualization:** Using Matplotlib, Seaborn, and Plotly to build interactive and static dashboards.
- **SQL Analytics:** Complex querying, aggregations, and window functions for massive databases.
